# esp32_mlx90640_uart_tx

## Environment

* PlatformIO
* WEMOS LOLIN32 Lite

## Pinout

* I2C
    * SDA pin 19
    * SCL pin 23
* Uart
    * Tx pin 17
    * Rx pin 16
