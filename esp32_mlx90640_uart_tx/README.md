# esp32_mlx90640_uart_tx

## Environment

* PlatformIO
* WEMOS LOLIN32 Lite

## Pinout

* I2C
    * GPIO 19 = SDA
    * GPIO 23 = SCL
* Uart
    * GPIO 17 = Tx
    * GPIO 16 = Rx
